# IEDC App

A cross platform mobile application for IEDC_CCE members

<p>&nbsp;</p>
<p>&nbsp;</p>
<p>&nbsp;</p>
<p>&nbsp;</p>
<p>&nbsp;</p>

# Objectives 

IEDC App aims to provide easy notifications and details of upcoming events, official notificationd and a reward system based on the achievements relevant to IEDC.
The members can claim these reward tokens by participating and winning competitions in and out of IEDC_CCE. These reward points than then be claimed for awesome merchantises.
